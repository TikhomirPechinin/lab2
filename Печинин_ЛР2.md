3.2 Создание проекта
Создан файл main.py:

python
def calculate_sum(numbers):
    total = 0
    for i in range(len(numbers)):
        total += numbers[i]
    return total / len(numbers)

nums = [10, 20, 30]
print(calculate_sum(nums))
